OA_Plan_BOM2 发布说明（简体中文）

一、文件说明
1) OA_Plan_BOM2_portable_CONSOLE.exe
   - 推荐使用
   - 运行时会保留黑色控制台窗口，可查看 print 日志和进度信息

2) OA_Plan_BOM2_portable_GUI.exe
   - 静默界面版
   - 运行时不显示控制台窗口

二、运行环境
1) Windows 10/11 64 位
2) 能访问 OA 与帆软系统（含公司网络/VPN）
3) 当前用户对运行目录有读写权限

三、使用步骤
1) 双击 OA_Plan_BOM2_portable_CONSOLE.exe（建议）
2) 在启动弹窗中填写参数：
   - OA/帆软账号和密码（可选记住）
   - 年份、月份、OA 工厂、帆软工厂
   - 可选本地 Excel：第一列从第二行开始是物料编码
3) 点击确认，等待程序自动执行
4) 观察控制台日志，确认每一步是否成功

四、输出结果
1) 程序会在运行目录生成下载与导出文件
2) 若选择本地 Excel，会跳过 OA 抓取，直接执行帆软导出

五、常见问题
1) EXE 被杀毒软件拦截
   - 将发布目录加入白名单后重试

2) 登录失败
   - 检查账号密码、网络/VPN、系统访问权限

3) 页面加载慢或导出超时
   - 在参数弹窗的“节拍配置”中适当调大等待时间

4) 浏览器启动失败
   - 当前发布包已内置便携版 Chrome 并自动定位
   - 若仍失败，请联系维护人员更新发布包

六、注意事项
1) 发布包不包含个人 runtime_config.json，避免账号信息泄露
2) 建议将发布包放在简短路径下运行（如 D:\BOM\）
